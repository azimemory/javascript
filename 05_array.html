배열

		자바스크립트에서 모든 타입의 데이터를 저장하는 자료구조
		자바의 list와 유사하다.

indexOf : 배열에서 요소가 위치한 인덱스를 반환

push : 배열의 맨 뒤에 요소 추가 / pop : 배열의 맨 뒤 요소를 제거

unshift : 배열의 맨 앞에 요소를 추가 / shift : 배열의 맨 앞에 요소를 제거

splice : 배열의 요소를 선택해서 잘라내거나 교체

		splice(idx,cnt,repalce)
			idx : 자르거나 교체 시작할 인덱스
			cnt : 자르거나 교체할 개수
			replace : 교체할 값

slice :원본 배열을 변경시키지 않고 배열의 요소를 지정해 복제하는 메서드

concat : 배열을 결합하여 반환하는 메서드

join : 배열의 요소들을 문자열로 반환

sort : 배열 정렬

reverse : 배열의 순서를 뒤집는 메서드

reduce() : 배열의 각 요소에 대해 주어진 콜백함수(reducer) 실행하고 결과값을 반환

		reduce(reducer,initValue) :
		
		reducer : 콜백함수
			reducer의 매개변수
				첫번째 매개변수 : 콜백함수에서 마지막으로 return된 값
				두번째 매개변수 : 현재 값 
				세번째 매개변수 : 현재 요소의 인덱스
				네번째 매개변수 : reduce()를 호출한 배열
	
		initValue : 시작값
					만약 없으면 배열의 첫번째 값을 시작값으로 사용한다
